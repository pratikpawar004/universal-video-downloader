🚀 Universal Video Downloader

This project is a multi-platform video downloader that allows users to download videos and audio from various platforms.

✨ Features

  • Download videos in multiple qualities (1080p, 720p, 360p)
  • Extract audio (MP3)
  • Supports multiple platforms
  • Clean UI with loading animation
  • Fast and reliable downloads using yt-dlp

📌 Supported Platforms

  • YouTube
  • Instagram
  • Facebook
  • Twitter / X
  • Vimeo
  • Dailymotion
  • M3U8 Streaming Sites
